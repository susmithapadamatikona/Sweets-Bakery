# 🧁 Sweet Crumbs Bakery

> *Freshly Baked Happiness Every Day*

A complete, premium, fully-responsive multi-page bakery & sweet-shop website built with **HTML5, CSS3 and Vanilla JavaScript only** — no frameworks, no build step, no backend.

## ▶️ How to run

Just open **`index.html`** in any modern browser. That's it.

> Product photos load from Unsplash (royalty-free) when you're online. If you're offline or an image fails, each one gracefully falls back to a warm gradient + bakery emoji, so the layout always looks intentional.

## 🎨 Design system

- **Palette:** warm pastel bakery tones (`#F9E1CF` → `#9B6B52`) driven by CSS custom properties
- **Fonts:** Playfair Display (headings) + Poppins (body)
- **Style:** soft rounded corners, organic blob backgrounds, glassmorphism, soft shadows

## ✨ Features

Dark-mode toggle · live product search + suggestions · category / price / rating filters · sort ·
wishlist (localStorage) · shopping cart with coupons & totals (localStorage) · quick-view modal ·
gallery masonry + lightbox · testimonial slider · FAQ accordion · animated counters · scroll reveals ·
custom cursor · ripple buttons · form validation · password strength meter · scroll-to-top · preloader ·
toast notifications · lazy-loaded images · fully responsive (1440 / 1024 / 768 / 480) · accessibility & SEO friendly.
